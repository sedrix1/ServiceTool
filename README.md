# Windows Service Tool

Current Version: v0.9

A lightweight Windows service management tool built with native batch scripting.

---

## Features

- Interactive CLI menu
- Services Status Monitoring
- Enable / Disable Services
- Basic Windows Performance Tweaks
- Lightweight and dependency-free
- Compatible with Windows 10 and Windows 11
- Simple batch scripting architecture

---

## Why This Project Exists

Managing Windows services manually through "services.msc" can be repetitive and time-consuming, especially for advanced users or gamers.

This project provides a faster and more scriptable alternative using native Windows batch scripting without requiring external software.

---

## Requirements

- Windows 10 / Windows 11
- Administrator Privileges

---

## Installation

git clone https://github.com/sedrix1/windows-service-tool.git
